Product Requirements Document: Flux Creative Technologies React Page

Project Overview

This project is a React-based landing page for Flux Creative Technologies, designed as part of an internship task. The goal is to build a responsive, reusable UI that adheres to the brand's visual identity (pink, purple, and blue accents on a dark tech background).

Components to Build

  1. Navbar: Responsive navigation with the Flux logo and links (Home, Services, Work, Pricing, Contact).

  2. Hero Section: A high-impact introduction featuring a bold headline, supporting subtext, and a primary call-to-action (CTA) button.

  3. Features or Services Section: A grid-based layout showcasing the core offerings (e.g., Web Design, Mobile App Development, Tech Consulting).

  4. Footer: Professional footer with logo, navigation links, social media icons, and contact information.

Design Guidelines

  • Color Palette: Dark theme (deep blacks/grays) with vibrant gradients of pink, purple, and blue.

  • Typography: Modern, clean sans-serif fonts.

  • Visual Style: Clean lines, subtle glassmorphism, and high-quality imagery or abstract graphics.

Technical Requirements

  • Framework: React

  • Styling: Tailwind CSS (recommended for speed and consistency)

  • Deployment: GitHub Pages

  • Documentation: Detailed README.md

README.md Structure

  • What was built: A responsive landing page for Flux Creative Tech.

  • Components Created: Navbar, Hero, Features, Footer.

  • How to Run:

    1. Clone the repo.

    2. npm install

    3. npm start

  • Deployment: Link to GitHub Pages.
