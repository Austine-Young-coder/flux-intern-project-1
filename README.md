# Flux Creative Technologies - Landing Page

This is a modern, dark-themed React landing page for Flux Creative Technologies, built as part of an internship task. The design features a sophisticated dark color scheme with vibrant gradients and glassmorphism effects.

## What I Built

I created a responsive React application with the following components:

- **Navbar**: Fixed navigation with Flux logo, menu links (Solutions, Technology, Showcase, About), and CTA button
- **Hero Section**: High-impact introduction with gradient text, dual CTAs, and animated abstract shapes
- **Core Spectrum Section**: Services showcase with three feature cards featuring icons and descriptions
- **Footer**: Multi-column footer with brand info, navigation links, newsletter signup, and legal links

## Design System

The application uses a custom dark design system with:
- **Primary**: #8B5CF6 (Purple)
- **Secondary**: #3B82F6 (Blue)
- **Tertiary**: #EC4899 (Pink)
- **Background**: #191919 (Dark)
- **Typography**: Inter font family
- **Effects**: Glassmorphism, gradients, and smooth animations

## Components Created

1. `Navbar.jsx` - Navigation component with logo and menu
2. `HeroSection.jsx` - Hero section with animated elements
3. `FeaturesSection.jsx` - Services/Core Spectrum section
4. `Footer.jsx` - Comprehensive footer with multiple columns

Each component has its own CSS file for styling.

## How to Run the Project

1. Clone the repository
2. Install dependencies: `npm install`
3. Start the development server: `npm run dev`
4. Open your browser to `http://localhost:5173`

## Technologies Used

- React 19
- Vite
- CSS3 with custom properties
- Modern CSS features (backdrop-filter, gradients, animations)

## Deployment

The project is set up for deployment on GitHub Pages.

### Steps to Deploy:

1. Create a new repository on GitHub
2. Push your code to the repository
3. Update `vite.config.js` with your repo name: `base: '/your-repo-name/'`
4. Run `npm run deploy` to deploy to GitHub Pages
5. Enable GitHub Pages in repository settings

Your site will be available at `https://your-username.github.io/your-repo-name/`

The React Compiler is not enabled on this template because of its impact on dev & build performances. To add it, see [this documentation](https://react.dev/learn/react-compiler/installation).
